# Product Context — NexusOps

## What It Does
NexusOps bridges building commissioning/closeout data into day-1 facility operations. It provides:
- **Operational Readiness Dashboard** — readiness scores per project based on issues, tests, closeout docs
- **Work Order Management** — full lifecycle from creation to verification with SLA tracking
- **IoT Sensor Monitoring** — real-time sensor data, threshold-based anomaly detection, auto work order creation
- **Asset Management** — registry with QR codes, health scoring, maintenance scheduling
- **Vendor Management** — contracts, scorecards, NTE pricing
- **Workflow Automation** — visual rule builder for triggers/conditions/actions

## Who Uses It
- **Facility Directors** — portfolio dashboard, readiness scores, executive KPIs
- **Commissioning Managers** — project detail, handover blockers, closeout tracking
- **Maintenance Technicians** — work orders, checklists, asset detail
- **Building Occupants** — public request portal (no auth), status tracking
- **Integration Admins** — API docs, sync status, automation rules

## User Experience Goals
- Feels like a premium SaaS product, not a CRUD app
- Data-dense but not cluttered — facility managers need information density with clarity
- Mobile-aware (responsive) though primarily desktop
- Real-time feel via Livewire polling (5-15s refresh cycles)
- Dark sidebar with emerald accent, Stitch-generated UI design language
